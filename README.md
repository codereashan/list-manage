## Getting Started

To get a local copy up and running, follow these steps:

1. **Clone the repository:**

   ```bash
   git clone https://github.com/your-username/list-manage.git
   cd list-manage

2. **Install Dependencies:**

   ```bash
   npm i

3. **Run Application:**
   ```bash
   npm run dev
